El gran orden del CSS

Tu CSS funciona... pero es un caos: clases que chocan, estilos que no se reutilizan y miedo a borrar nada. Tu misión: ponerlo en orden aplicando tres metodologías. Tres retos, una metodología en cada uno. Al final, cada uno crea su propio repositorio y publica su sitio con GitHub Pages.

Cómo funciona

Resuelve los tres retos por separado (no mezcles metodologías). Cada reto tiene su escenario, su lista de requisitos y una pista conceptual. Cuando termines, sigue la página de Entrega para subirlo a GitHub Pages.

Cada reto trae un checklist con barra de progreso que guarda tu avance en el navegador. Úsalo para no dejarte nada.
